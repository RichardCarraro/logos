# LOGOS: Jornada Bíblica 📖✨

**Logos** é uma plataforma cristã interativa e gamificada, projetada para transformar o estudo das Escrituras em uma jornada envolvente de aprendizado e descoberta. Com foco em UI/UX minimalista e profissional, o app combina jogos de lógica, trilhas de conhecimento e um sistema de progressão recompensador.

---

## 🚀 Funcionalidades Principais

### 🏹 Desafios de Palavras (Termo Bíblico)
Um jogo de adivinhação inspirado no Wordle, com termos específicos das Escrituras:
*   **Modo Termo**: Uma palavra por dia.
*   **Modo Dueto**: Duas grades simultâneas para um desafio maior.
*   **Modo Quarteto**: Quatro palavras ao mesmo tempo, testando sua agilidade mental e conhecimento bíblico.
*   *Layout 100% adaptativo e sem scroll, garantindo foco total no jogo.*

### 📚 Manual do Discípulo (Aprenda as Escrituras)
Uma enciclopédia interativa dividida em 12 trilhas de estudo:
*   **Antigo Testamento**: Da Criação aos Profetas.
*   **Novo Testamento**: Do nascimento de Cristo à revelação final.
*   **Personagens e Parábolas**: Heróis da fé, mulheres bíblicas e os ensinamentos de Jesus.
*   **Mini-Quizzes**: Valide seu aprendizado ao final de cada lição para ganhar XP.

### 🎮 Gamificação e Progresso
Sistema de RPG para incentivar a constância nos estudos:
*   **XP e Níveis**: Ganhe pontos ao jogar e concluir lições. Suba de nível até se tornar um Mestre das Escrituras.
*   **Dias de Sequência (Streak)**: Contador funcional que incentiva o acesso diário à Palavra.
*   **Conquistas (Badges)**: Selos colecionáveis para marcos importantes na sua jornada.
*   **Persistência**: Todo o seu progresso é salvo localmente no navegador.

### 🎨 Design e UI/UX
*   Estética profissional e sóbria, utilizando paletas de cores que remetem à calma e ao estudo (Verde Musgo e Dourado).
*   Animações suaves com **Motion**.
*   Totalmente responsivo: jogue no celular, tablet ou computador.

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido com as tecnologias mais modernas do ecossistema web:

*   **React 18** + **Vite**
*   **TypeScript** (Segurança de tipos)
*   **Tailwind CSS** (Estilização utilitária)
*   **Lucide React** (Ícones elegantes)
*   **Motion** (Animações de alta performance)
*   **React Router** (Navegação SPA fluida)

## 📦 Como Executar o Projeto

1.  Clone o repositório:
    ```bash
    git clone https://github.com/seu-usuario/logos-jornada-biblica.git
    ```
2.  Acesse a pasta do projeto:
    ```bash
    cd logos-jornada-biblica
    ```
3.  Instale as dependências:
    ```bash
    npm install
    ```
4.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
5.  Abra o navegador em `http://localhost:3000`.
